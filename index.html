Summary

의료영상 소프트웨어 개발자로서 C++/Qt 기반 DICOM 클라이언트, PACS 연동, AI 분석 결과 기반 리포트 생성, 매크로 자동화 기능을 구현해 왔습니다. DICOM/mask/AI 결과를 정규화해 PDF/CSV 리포트로 변환하고, Action/Command 기반 자동화 파이프라인으로 실제 의료 현장의 반복 처리·리포트 흐름을 안정화하는 데 집중해 왔습니다.

의료용 애플리케이션 설계 및 개발

C++/Qt 기반 DICOM 클라이언트에서 영상 로드, 분석 결과 표시, 3D View, HU 값 기반 정보 표시 등 의료영상 뷰어의 주요 기능을 개발·개선했습니다.

AI 결과 리포트 생성 파이프라인

DICOM 메타데이터, segmentation mask, 전달받은 AI 분석 결과를 JSON·CSV로 정규화하고 QWebEngine HTML 템플릿을 PDF로 렌더링하는 리포트 흐름을 구현했습니다.

매크로 자동화 아키텍처

Command 계열 클래스와 Action Queue를 기반으로 DICOM 로드, AI 결과 처리, layer·mesh export, 리포트 저장 단계를 분리해 순차 실행 구조를 구성했습니다.

DCMTK 기반 DICOM/PACS 연동

DCMTK를 활용해 DICOM 파일 송수신, DICOM Echo, PACS 조회·다운로드·업로드, 분석 결과 이미지 저장과 업로드 흐름을 구현했습니다.

테스트 자동화와 제품 안정화

UI Auto Test, mipAutoPilot Tester, TeamCity 연동, 팝업 제어, PACS 자동 테스트를 통해 반복 검증 체계를 확장하고 장시간 실행 안정성을 개선했습니다.

배포·운영·고객/기관 대응

Alpha/Beta/RC/Hotfix 배포, 릴리즈 노트, 밸리데이션 문서, 기관별 리포트 요구사항, 고객 이슈 분석과 원격 지원까지 제품 출시 이후 단계도 담당했습니다.
